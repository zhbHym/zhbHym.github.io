PHP环境搭建

2018-11-11   PHP安装 

1、所用到的软件

由于PHP所用到的软件较多，单独安装比较繁琐，因此这里使用 WAMP 集成软件和 eclipse for PHP Developers ,在 WAMP 中包含有MysQL、PHP、Apache以及XDebug。

WAMP的下载地址为：http://www.wampserver.com/en/#download-wrapper
下载默认安装就可以。

eclipse for PHP Developers 下载地址为：http://www.eclipse.org/downloads/packages/eclipse-php-developers/mars1
下载后解压便可以使用（当然配置了JDK）。

2、配置 WAMP

（1）Apache的配置

通常情况下，80端口很容易被占用，因此这里改80端口为8088端口：

在wamp安装目录下的 bin\apache\apache2.4.9\conf 文件夹下，打开httpd.conf文件夹，找到Listen，将ip地址后的80改为8088（当然你也可以改成其他端口），这时重启wamp,地址栏输入localhost:8088就会出现首页。但是phpMyAdmin、SQLiteManager，依旧是默认的80端口，打开wamp安装目录下的wampmanager.tpl文件，在http://localhost后面添加 ：8088，保存并退出，重启wamp后生效。

修改www目录，一般情况下，我是不喜欢把软件与代码放在一起，显得很乱，因此更改到自己的路径下，当然，你也可以不设置，如果不设置，则创建的project必须放在wamp安装路径下的www文件夹中。

① 打开bin\apache\apache2.4.9\conf目录下的 httpd.conf 文件夹，分别搜索 DocumentRoot "d:/wamp/www/"和 <Directory "d:/wamp/www/">,然后更改 d:/wamp/www/ 为自己的路径，例如我的是 D:/Project/PHPproject/,
② wampmanager.tpl文件下，修改 FileName: "${wwwDir}"为 FileName:"D:/Project/PHPproject"
③ wampmanager.ini文件，修改如下:[Menu.Left] 下FileName: “d:/wamp/www” 修改为FileName: “D:/Project/PHPproject”。
④ 将wamp安装路径下www文件夹下的文件拷贝到你修改后的路径下，并将index.php文件中的 $wampConfFile = $server_dir.'wampmanager.conf';更改为 $wampConfFile = 'D:/wamp/wampmanager.conf';(自己的实际路径)，重启wamp ,可以看到配置完成。

（2）Xdebug 配置

现在的版本中已经包含了Xdebug，在安装路径下的 bin\php\php5.5.12\zend_ext目录下，并且已经配置好了，我们唯一要做的就是在 bin\apache\apache2.4.9\bin 目录下的 php.ini中的

xdebug.remote_enable = off ，改成 xdebug.remote_enable = on 。这样做是为了在eclipse中调试时可以进入断点。

3、创建项目

配置项目路径

打开eclipse，workspace设置为自己在Apache中设置的路径，未设置选择wamp安装路径下的www文件夹。

配置调试环境

进入Eclipse – Window – Preferences 设置 PHP – PHP Executable。点击右侧的add，弹出的Executable path选择wanp安装路径下的bin\php\php5.5.12\php.exe文件,php ini file路径选择bin\php\php5.5.12\php.ini。

设置PHP–>debug,选择XDebug就ok了。

点击file – new – php project ,设置项目，创建完成。

然后点击项目名称一次选择 new – PHP file,设置文件名connsql，创建php文件

<?php
// 连接数据库，地址、账号、密码
$con = mysqli_connect("localhost:3306","root","") or die("error:数据库连接失败".mysqli_error());
// 设置表明
mysqli_select_db($con,"test");

?>

再创建userlogin.php文件，编写如下代码：

<?php
session_start();
header("content-type:application/json");
$action = $_GET['action'];

if($action == "login"){
    $name = stripslashes(trim($_GET['name']));
    $pass = stripslashes(trim($_GET['pass']));
    $jsons = array();

    if ($name != '' &&  $pass != '')
    {
        require_once ('connsql.php');
        $query = mysqli_query($con,"SELECT id,name,pwd  FROM user WHERE name = '$name' and pwd = '$pass' LIMIT 1;");
        if ($result = mysqli_fetch_array($query)){
            //echo "登录成功";
            $data = array();
            $data['id'] = $result['id'];
            $data['name']=$result['name'];
            $data['pass']=$result['pwd'];

            $jsons['state'] = 0;
            $jsons['data'] = $data;
            $jsons['msg'] = "";
            print "对";
        }else {
            print "错";
            $jsons['state'] = 1;
            $jsons['msg'] = "用户不存在";
        }
    }

    echo json_encode($jsons);
}

点击run按钮，选择php web application,
然后再浏览器中输入
http://localhost:8088/test/PHP/userlogin.php?action=login&name=tom&pass=123
返回json字符串表示测试成功。
